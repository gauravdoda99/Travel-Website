
# 🌍 TravelX - Responsive Travel Website

## 📖 Overview

**TravelX** is a modern, responsive, and visually appealing travel website developed using **HTML5** and **CSS3**. The project is designed to provide users with an engaging experience while exploring popular travel destinations, tour packages, company information, and contact details.

The website features a clean interface, smooth animations, responsive layouts, and attractive travel-themed images. It is fully optimized for desktop, tablet, and mobile devices.

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Ensure all images are placed inside the **images** folder.
4. Open **index.html** in your browser.
5. Enjoy exploring the website.
